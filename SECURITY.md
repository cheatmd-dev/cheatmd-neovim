# Security Policy

cheatmd.nvim runs the local `cheatmd` command-line tool for linting and interactive execution.

Please report security issues privately by opening a GitHub security advisory for this repository:

https://github.com/cheatmd-dev/cheatmd-neovim/security/advisories/new

Include:

- Plugin version or commit
- Neovim version
- Operating system
- Steps to reproduce
- Relevant logs or error messages, with private workspace content removed
